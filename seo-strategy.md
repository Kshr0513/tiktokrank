# SEO戦略 — 「Twitter保存ランキング」検索流入の獲得

## 目的

「Twitter保存ランキング」を検索するユーザー層に本サイトを発見してもらい、「TikTok版を探していた層」を取り込む。

## 重要な前提(リスク管理)

- **「Twitter保存ランキング」を自サイト名にしない・なりすまさない。** 他サービス名での偽装はGoogleのスパムポリシー違反・商標トラブルの原因になり、ペナルティで圏外に飛ぶリスクが最も高い
- 戦い方は「**関連サイト・比較コンテンツとしての共起**」。「Twitter保存ランキングのTikTok版」「Twitter保存ランキングと似たサービス」という文脈で言及し、関連検索・指名検索の周辺ワードを取りに行く
- ダウンロードを誘導する文言(「保存方法」「ダウンロードする方法」をサイト機能として謳う)は書かない。あくまで「人気動画ランキング」「リンク先のTikTokで見る」と表現する

## キーワード設計

### Tier 1(指名系・最優先)

- `Twitter保存ランキング TikTok`
- `Twitter保存ランキング TikTok版`
- `TikTok保存ランキング`
- `ティックトック 保存ランキング`

### Tier 2(一般ワード)

- `TikTok 人気動画 ランキング`
- `TikTok バズ動画 今日`
- `TikTok ランキング リアルタイム`

### Tier 3(ロングテール / 個別動画ページで取る)

- 動画タイトル・投稿者名を含む検索(`/video/[id]` ページが受け皿)

## サイト内施策

### 1. タイトル・メタ設計

- トップ: `TikTok保存ランキング | Twitter保存ランキングのTikTok版 - 今バズってる動画`
- description: 「Twitter保存ランキングのようにTikTokの人気動画をリアルタイム集計。今みんなが注目している動画がわかります。」
- 各期間ページ: `【{今日の日付}】TikTok 24時間人気ランキング` のように日付を動的挿入(鮮度シグナル)

### 2. 構造化データ(JSON-LD)

- ランキングページ: `ItemList`(各動画を `ListItem` で順位付き)
- 個別動画ページ: `VideoObject`(thumbnailUrl, uploadDate, author を oEmbed から)
- サイト全体: `WebSite` + `SearchAction`
- about: `FAQPage`(「Twitter保存ランキングとの違いは?」「動画は保存できますか?→当サイトでは保存機能を提供していません」等のQ&A。指名ワードを自然に含められる)

### 3. コンテンツページ

`/about` とブログ的な解説ページ(`/guide/...`)で指名ワードの共起を作る:

- 「Twitter保存ランキングとは?TikTok版である当サイトとの違い」
- 「TikTokでバズっている動画の探し方」
- 比較・解説記事はseo-writerエージェントで生成し、人間がファクトチェックしてから公開

### 4. 技術SEO

- 全ランキングページISRでHTML完結(JSオフでも順位が読める状態)
- `sitemap.xml` 自動生成(個別動画ページを含む。更新頻度の高いランキングページは priority 高め)
- `robots.txt`、canonical、OGP/Twitterカード(SNSシェア時のCTR向上)
- Core Web Vitals: サムネイルは `next/image` + 遅延読み込み、CLS対策で広告枠は固定サイズ

### 5. 鮮度シグナル

- ランキングは60秒ISRで常に更新 → クロール頻度向上
- 個別動画ページに「順位推移」を出すことでページが定期更新される構造にする

## サイト外施策(運営者タスク、コードでは対応しない)

- X(Twitter)公式アカウントで毎日ランキングを自動投稿(指名ワード入り)→ 指名検索とサイテーション獲得
- Google Search Console / Bing Webmaster 登録、インデックス登録リクエスト
- 「Twitter保存ランキング 代わり」「類似サービス」系のまとめ記事への掲載打診

## 計測

- Search Console でTier 1ワードの掲載順位を週次トラッキング
- GA4: 投稿コンバージョン(リンク投稿完了)とTikTok遷移クリックをイベント計測

## 広告との両立に関する注意

主要広告ネットワーク(Google AdSense等)はアダルトコンテンツを禁止している。Twitter保存ランキング系サイトの流入はアダルト寄りに偏りやすいため、**モデレーション(NGワード+通報+非表示)を機能させて一般コンテンツ中心のランキングを維持すること**が、広告審査通過とSEO評価の両方の前提になる。アダルトに振る場合は一般広告ネットワークは使えず、SEO上もセーフサーチで除外されるため戦略全体が変わる。v1は全年齢向けで設計する。
